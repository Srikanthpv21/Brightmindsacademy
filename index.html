Course Syllabus: Programming for Future Innovators

A detailed 8-week schedule for beginners.

Week 1: The Foundation of Logic & Problem Solving

Goal: Understand how programmers think and plan solutions before writing code.

Session 1: Intro to Programming & Algorithms. Activity: Write the step-by-step algorithm for a daily task.

Session 2: Visualizing Logic with Flowcharts & Pseudocode. Activity: Create a flowchart to decide whether to carry an umbrella.

Weekly Project: Create a flowchart for your complete morning routine.

Week 2: Introduction to Python

Goal: Write your first Python programs and handle user input and data.

Session 3: Your First Code & Variables. Activity: Store your name in a variable and print a greeting.

Session 4: Interacting with the User. Activity: Create a program that adds two numbers given by the user.

Weekly Project: Build a "Bio Generator" that creates a summary from user input.

Week 3: Logic and Control Flow in Python

Goal: Learn to make decisions and repeat actions in your programs.

Session 5: Making Decisions with `if` Statements. Activity: Code a program that checks if a user is old enough to vote.

Session 6: Repeating Actions with Loops. Activity: Use a `for` loop to print numbers 1 to 10.

Weekly Project: Create a number guessing game with "too high" / "too low" hints.

Week 4: Project Week - Python Text-Based Game

Goal: Combine all your Python skills to build a complete, interactive game.

Session 7: Functions & Project Planning. Activity: Write a simple function that greets a user.

Session 8: Project Building Workshop & Debugging. Activity: Build the first part of the adventure game together.

Weekly Project: Finish and submit your complete Python Text-Based Adventure Game.

Week 5: Building the Webâ€™s Skeleton with HTML

Goal: Understand how websites are built and create the structure for your own webpage.

Session 9: Intro to HTML & Essential Tags. Activity: Create your first `index.html` file with a heading and paragraph.

Session 10: Structuring Content with Images, Links & Lists. Activity: Add a picture and a link to your page.

Weekly Project: Build a 2-page "About Me" website with a linked "Hobbies" page.

Week 6: Styling the Web with CSS

Goal: Learn to apply colors, fonts, and layouts to make your website beautiful.

Session 11: Intro to CSS & Styling Text. Activity: Change the colors and fonts of your website.

Session 12: The Box Model & Layout with Flexbox. Activity: Center your content and arrange it in columns.

Weekly Project: Fully style your 2-page website using CSS.

Week 7: Making Websites Interactive with JavaScript

Goal: Use JavaScript to make your website respond to user actions.

Session 13: Intro to JavaScript & the DOM. Activity: Select a page element with JS and log it to the console.

Session 14: Handling Events & Manipulating Content. Activity: Create a button that changes text on the page.

Weekly Project: Add a "dark mode" toggle button to your website.

Week 8: Capstone Project & Wrap-Up

Goal: Integrate all your web skills to build a final, impressive interactive application.

Session 15: Capstone Project Planning Workshop. Activity: Outline the features for your chosen capstone project.

Session 16: Final Q&A and Next Steps. Activity: Showcase your project progress and get feedback.

Final Project: Complete and submit your capstone web application (e.g., To-Do List, Interactive Quiz).
